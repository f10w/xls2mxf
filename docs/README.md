# XLS2MXF

Инструмент для подготовки эфирных рекламных блоков из траффик-листов.

Работает в двух режимах:

- **Ручной** — извлекает ID роликов из траффик-листа за нужную дату, находит
  соответствующие `.mxf` файлы и копирует их в отдельную папку. Опционально
  кладёт список ID в буфер обмена для ручной вставки в Excel.
- **Автоматический** — нарезает траффик-лист на рекламные блоки, для каждого
  блока склеивает через ffmpeg последовательность «открывашка → ролики → закрывашка»
  в готовый эфирный файл, проверяет хронометраж и сам устраняет типовые проблемы
  (нестандартное аудио, недостающие ролики).

---

## Быстрый старт

1. Положите рядом с `copy_rollers.exe` файл `copy_rollers.conf` (при первом запуске
   он создастся автоматически со значениями по умолчанию — отредактируйте пути).
2. Убедитесь, что `ffmpeg.exe` и `ffprobe.exe` доступны: либо в `PATH`, либо рядом
   с программой, либо пропишите пути в conf (секция `[ffmpeg]`).
3. Запустите. Без аргументов программа спросит режим и дату интерактивно.

```
copy_rollers.exe
```

### Типовые команды

Проверить смену перед сборкой (быстро, ничего не собирает):
```
copy_rollers.exe --check --date 170626
```

Автоматическая сборка эфира на конкретную дату:
```
copy_rollers.exe --mode auto --date 170626
```

Ручной режим (копирование роликов в папку):
```
copy_rollers.exe --mode manual --date 170626
```

Если `--date` не указан — программа предложит ввести дату, по умолчанию завтрашнюю.

---

## Аргументы командной строки

| Аргумент | Значение |
|----------|----------|
| `--date ДДММГГ` | Дата смены. Без неё — интерактивный ввод (дефолт: завтра). |
| `--mode manual\|auto` | Режим. Без него — программа спросит (1 — ручной, 2 — авто). |
| `--check` | Dry-run: проверить готовность смены без сборки и перекодов. |
| `--xlsx ПУТЬ` | Папка с траффик-листами (переопределяет conf). |
| `--src ПУТЬ` | Папка с роликами `.mxf` (переопределяет conf). |
| `--dst ПУТЬ` | Папка назначения (переопределяет conf). |

---

## Что создаётся

- **Ручной режим:** папка `ролики на ДДММГГ` (внутри `dst`) со скопированными `.mxf`.
- **Авто-режим:** папка `эфир на ДДММГГ` (внутри `dst`) с собранными файлами вида
  `ДД-ММ-ГГ_Reklama_RTR_ЧЧ-ММ.mxf` — по одному на рекламный блок.
- **Лог** `ДДММГГ.log` рядом с программой — подробности прогона.

---

## Документация

- **CONFIG.md** — справочник по всем параметрам `copy_rollers.conf`.
- **RUNBOOK.md** — пошаговый сценарий эфирной смены.
- **ARCHITECTURE.md** — внутреннее устройство для доработки кода.

---

## Требования

- Windows 10 или новее.
- `ffmpeg` и `ffprobe` (для авто-режима).
- Траффик-листы в формате `.xlsx` с датой `ДДММГГ` в конце имени файла.
