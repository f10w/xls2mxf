BUILDING xls2mxf.exe (Windows 10+)
======================================

1. Install Python from https://www.python.org/downloads/
   IMPORTANT: check "Add Python to PATH" during installation.

2a. For a development build — double-click build.bat.
    The finished xls2mxf.exe will appear in the dist\ folder.

2b. For a release build — double-click release.bat.
    Reads the version from xls2mxf/__init__.py, builds the exe, and packages it
    into releases\xls2mxf_v{VERSION}\ plus a zip archive alongside it.

4. Copy xls2mxf.exe from dist\ to wherever you need it.
   Place next to it:
     - xls2mxf.conf  (auto-created on first run; fill in the paths afterwards)
     - ffmpeg.exe, ffprobe.exe  (if they are not already in PATH)

USAGE
=====
  xls2mxf.exe                               — prompts for mode and date
  xls2mxf.exe --check --date 170626         — verify session without assembly
  xls2mxf.exe --mode auto --date 170626     — auto-assemble broadcast blocks
  xls2mxf.exe --mode manual --date 170626   — copy clips to folder

See the docs\ folder for details:
  README.md        — overview and quick start
  CONFIG.md        — all conf parameters
  RUNBOOK.md       — broadcast session guide
  ARCHITECTURE.md  — code internals (for developers)
