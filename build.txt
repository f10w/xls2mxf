СБОРКА xls2mxf.exe (Windows 10+)
======================================

1. Установить Python с https://www.python.org/downloads/
   ВАЖНО: при установке отметить галочку "Add Python to PATH".

2. Запустить build.bat двойным кликом.
   Он сам поставит pyinstaller и openpyxl и соберёт exe.

3. Готовый xls2mxf.exe появится в папке dist\

4. Достать xls2mxf.exe из dist\, положить куда нужно.
   Рядом с ним:
     - xls2mxf.conf  (создастся сам при первом запуске, потом впишите пути)
     - ffmpeg.exe, ffprobe.exe  (если их нет в PATH)

ЗАПУСК
======
  xls2mxf.exe                      — спросит режим и дату
  xls2mxf.exe --check --date 170626   — проверка смены без сборки
  xls2mxf.exe --mode auto --date 170626 — автосборка эфира
  xls2mxf.exe --mode manual --date 170626 — копирование роликов

Подробности — в папке docs\:
  README.md        — обзор и быстрый старт
  CONFIG.md        — все параметры conf
  RUNBOOK.md       — сценарий эфирной смены
  ARCHITECTURE.md  — устройство кода (для доработки)
