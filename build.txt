BUILDING xls2mxf.exe (Windows 10+)
======================================

1. Install Python from https://www.python.org/downloads/
   IMPORTANT: check "Add Python to PATH" during installation.

2. Double-click build.bat.
   It will install pyinstaller and openpyxl automatically, then build the exe.

3. The finished xls2mxf.exe will appear in the dist\ folder.

4. Copy xls2mxf.exe from dist\ to wherever you need it.
   Place next to it:
     - xls2mxf.conf  (auto-created on first run; fill in the paths afterwards)
     - ffmpeg.exe, ffprobe.exe  (if they are not already in PATH)

USAGE
=====
  xls2mxf.exe                               — prompts for mode and date
  xls2mxf.exe --check --date 170626         — verify session without assembly
  xls2mxf.exe --mode auto --date 170626     — auto-assemble broadcast blocks
  xls2mxf.exe --mode manual --date 170626   — copy clips to folder

BUILDING ON LINUX
=================
  chmod +x build_linux.sh
  ./build_linux.sh

  The finished xls2mxf binary will appear in the dist/ folder.
  Copy it wherever needed and place xls2mxf.conf next to it.
  ffmpeg and ffprobe must be in PATH or placed next to the binary.

BUILDING ON macOS
=================
  chmod +x build_mac.sh
  ./build_mac.sh

  The finished xls2mxf binary will appear in the dist/ folder.
  Copy it wherever needed and place xls2mxf.conf next to it.
  Install ffmpeg via Homebrew (brew install ffmpeg) or place it next to the binary.

  Note: macOS may block unsigned binaries on first run.
  If you see a security warning, go to:
    System Settings -> Privacy & Security -> Allow Anyway
